작동 원리

매일 전화 한 통에서, 생명을 지키는 자동 대응까지.

영실이는 단순한 안부 전화가 아닙니다. 대화를 분석하고, 위험을 점수화하고, 필요한 순간 사람에게 연결하는 4단계로 작동합니다.

01

매일 안부 전화

영실이가 정해진 시간에 어르신께 직접 전화를 걸어 자연스럽게 안부를 묻습니다. 어르신은 받기만 하면 됩니다.

02

대화 속 위험 신호 감지

음성 대화를 실시간 분석해 건강·정서 신호를 정상·관찰·위험 3단계로 자동 분류합니다.

03

고독사 위험도 자동 분석

통화 기록과 미응답·이상 신호를 누적해 어르신별 위험도를 점수화하고 고위험군을 가려냅니다.

04

즉시 연결·대응

위험이 감지되면 망설임 없이 보호자·119·복지사에게 자동 연결해 골든타임을 지킵니다.

실제 통화 화면

영실이는, 이렇게 대화합니다.

딱딱한 기계음이 아닙니다. 천천히 듣고, 잘 못 알아들으면 다시 여쭤보고, 위험 신호는 놓치지 않습니다.

핵심 기능

대화의 온도를 읽고, 위기의 순간을 놓치지 않습니다.

사람의 귀로는 흘려보낼 수 있는 말 한마디를, 영실이는 일관된 기준으로 분류하고 기록합니다.

3단계 위험 키워드 분류

대화 속 표현을 정상·관찰·위험으로 나눠 일관되게 기록하고, 평소 상태(Baseline)와 비교합니다.

정상 · "괜찮아" 관찰 · "허리가 아파" 위험 · "쓰러졌어"

위험어 즉시 감지 → SOS

“살려줘”, “도와줘”, “쓰러졌어” 같은 위험 단어가 들리면 통화 중 즉시 긴급 대응 절차로 전환합니다.

보호자 자동 연결 119 복지사

스마트 안전 멘트 (기상청 연동)

폭염·한파·호우 특보가 발령되면, 해당 지역 어르신께 보내는 영실이의 안내 멘트가 자동으로 바뀝니다.

폭염한파호우

고독사 위험도 리포트

누적된 통화 데이터로 어르신별 위험 추이를 시각화해, 누가 더 자주 살펴야 하는지 알려줍니다.

보호자·복지사 안심 대시보드

오늘의 건강 상태, 통화 성공 여부, 위험 알림을 한 화면에서 확인합니다. 멀리 있어도 함께 돌봅니다.

고위험 지역 자동 식별

공공데이터를 분석해 돌봄이 시급한 지역과 가구를 자동으로 표시, 복지 자원을 가장 필요한 곳에 배치합니다.

날씨가 위험해질 때,
영실이의 말도 달라집니다.

두 가지 안부 방법

전화가 부담되면, 버튼 하나면 충분합니다.

어르신마다 익숙한 방법은 다릅니다. 영실이는 전화 통화와 앱 간편 체크, 두 가지로 매일의 안부를 확인합니다. 어떤 방법이든 결과는 자동으로 저장되고 같은 기준으로 분석됩니다.

AI 안부 전화

영실이가 직접 전화를 걸어 대화로 안부를 확인합니다. 받기만 하면 됩니다.

앱 간편 건강체크

전화가 어려운 분은 앱을 열어 오늘 컨디션을 버튼 하나로 선택합니다. 결과는 자동으로 서버에 저장돼 보호자·복지사에게 바로 전달됩니다.

버튼 하나의 선택도 정상·관찰·위험 3단계로 자동 분류되어, 전화와 똑같이 위험을 감지합니다. 이렇게 간단한 방식만으로도 매일의 관리가 가능합니다.
